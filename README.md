# ATP Tennis Match Prediction Model

A machine learning pipeline that predicts professional tennis match outcomes using 40 years of ATP data (1985–2024), custom ELO ratings, and XGBoost.

Inspired by [@theGreenCoding](https://x.com/theGreenCoding)'s viral project. Built from scratch using [Jeff Sackmann's tennis_atp dataset](https://github.com/JeffSackmann/tennis_atp).

## How It Works

1. **Data**: Every ATP match from 1985–2024 (~95k matches)
2. **ELO System**: Custom chess-style rating system adapted for tennis, with surface-specific ratings (Hard, Clay, Grass)
3. **Feature Engineering**: 43 pre-match features including ELO diff, head-to-head records, rolling win rates, serve stats, player attributes
4. **Model**: XGBoost classifier trained with strict no-data-leakage discipline

## Model Performance

| Model | Accuracy | ROC-AUC | Log Loss |
|-------|----------|---------|----------|
| Decision Tree | 64.6% | 0.703 | 0.623 |
| Random Forest | 64.5% | 0.716 | 0.614 |

## Quick Start

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Download data
```bash
mkdir -p data
cd data
for year in $(seq 1985 2025); do
  curl -sL "https://raw.githubusercontent.com/JeffSackmann/tennis_atp/master/atp_matches_${year}.csv" -o "atp_matches_${year}.csv"
done
curl -sL "https://raw.githubusercontent.com/JeffSackmann/tennis_atp/master/atp_players.csv" -o "atp_players.csv"
for decade in 80 90 00 10 20; do
  curl -sL "https://raw.githubusercontent.com/JeffSackmann/tennis_atp/master/atp_rankings_${decade}s.csv" -o "atp_rankings_${decade}s.csv"
done
cd ..
```

### 3. Train the model
```bash
python main.py
```

### 4. Predict a match
```bash
python predict_match.py "Jannik Sinner" "Carlos Alcaraz" "Hard"
python predict_match.py "Novak Djokovic" "Rafael Nadal" "Clay" --best_of 5 --round F --level G
```

## Updating with New Data

Jeff Sackmann's [tennis_atp repo](https://github.com/JeffSackmann/tennis_atp) is updated regularly. To refresh:

```bash
# Re-download latest CSVs
cd data && for year in $(seq 1985 2025); do
  curl -sL "https://raw.githubusercontent.com/JeffSackmann/tennis_atp/master/atp_matches_${year}.csv" -o "atp_matches_${year}.csv"
done && cd ..

# Retrain
python main.py
```

## Key Features

- **Custom ELO**: Overall + surface-specific ratings for every player since 1985
- **No Data Leakage**: All features computed using only pre-match information
- **P1/P2 Randomization**: Winner/loser columns randomized to prevent classifier bias
- **Head-to-Head**: Career H2H records computed for every matchup
- **Rolling Stats**: Win rates over last 10/25/50/100 matches
- **Serve Metrics**: Rolling averages of 1st serve win %, break points saved %, ace %
